# UPASANA - Integrated Yoga, Meditation & Cultural Education Centre

A world-class landing page for UPASANA, where ancient wisdom meets modern transformation. This project showcases a sanctuary dedicated to preventive healthcare, holistic development, and the preservation of India's timeless wisdom traditions.

## About UPASANA

UPASANA is a registered NGO committed to:
- **Education**: Structured meditation and cultural education programs
- **Preventive Healthcare**: Yoga, meditation, and holistic wellness practices
- **Cultural Preservation**: Hindustani classical music and Indian arts training

### Governance & Compliance
- ✓ Registered NGO
- ✓ 12A & 80G Tax Compliant
- ✓ CSR-1 Registered
- ✓ Audited Financial Statements Available

## Features

This landing page includes:
- Responsive, professional design with sophisticated typography
- **Video background in hero section** (with image fallback)
- 10 comprehensive sections covering mission, programs, and impact
- Interactive component showcase with tabbed navigation
- Smooth scroll animations and custom cursor effects
- Donation tiers and CSR partnership information
- Sanskrit wisdom integration with modern design aesthetics

## Quick Start

### Adding the Hero Video Background

The hero section supports a video background. To add it:

1. **Easy Method (Windows):**
   ```bash
   # Double-click download-video.bat
   # Or run in terminal:
   download-video.bat
   ```

2. **Manual Method:**
   - See detailed instructions in [VIDEO-SETUP.md](VIDEO-SETUP.md)
   - Download video from YouTube link
   - Save as `hero-video.mp4` in `video` folder
   - Refresh the page

3. **Without Video:**
   - The page works perfectly without the video
   - A beautiful static image will be used as fallback

## Installation


## Usage

Simply open `index.html` in your web browser to view the landing page. No build process or dependencies required.

```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```
